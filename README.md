# Reestar
Reestar is a simple static web page built using HTML and basic CSS. The page presents a food menu along with contact details, focusing on clean layout and beginner-friendly design.

## Page Highlights

Food menu section displaying available dishes

Contact details section for easy communication

Simple and neat UI using basic CSS

## Technologies Used

HTML5

CSS3

## Purpose

This project was created to practice front-end fundamentals, including HTML structuring and basic CSS styling, without using any frameworks or JavaScript.
